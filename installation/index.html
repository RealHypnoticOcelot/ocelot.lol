Installation

If you're launching through MultiMC and it's crashing, make sure your Java is up to date. More information: https://github.com/MultiMC/Launcher/wiki/Using-the-right-Java

If you're launching through the official Minecraft launcher, install Fabric at https://fabricmc.net/.

Once you have the correct version of Fabric installed, copy over the options and option.amecsapi files to your Minecraft folder. Do the same with the mods and config folder if you don't already have a folder with that name there.

If you do already have a mods and config folder, then merge the existing config folder with the new config folder.
Then, check your mods folder. If it's empty, then just replace it with the new mods folder. If it has files within it, create a sub-folder within, and move the files there. Then, merge the new and old mods folder. When you launch your Fabric instance, the mods should be loaded and configured!


If you are using MultiMC, the process is basically the same. Click on "Edit Instance", and if you haven't already, install Fabric onto the instance.

Once you've configured that, click on "Open .minecraft", and then just copy the folders and files in, like how you would normally.
